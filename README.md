# DynamiteSchedules
Web Programming Final for Spring 2026  
Developed by: Kay Perschka, Megan Edder, and Everett Walker

Dynamite Schedules is a schedule builder web application built with Node.js and MongoDB

SCHEDULE BUILDER  
Users can search and select courses from the database to build a personalized schedule. The system automatically validates prerequisites before a course can be added. Once selections are complete, users review their chosen courses in a cart, confirm them, and generate a finalized schedule.

COURSE CATALOG  
The full course catalog is browsable by department. Users select a department to view all available courses, which are rendered as individual course cards (built with React) displaying the course name and a short description.

RESOURCES  
The Resources page provides students with links to registrar documents, faculty profile pages, and university tutoring options.

ADMINISTRATOR PANEL  
Administrator accounts have elevated permissions and access to a dedicated management page. Admins can:

View and manage student profiles
Add or remove courses from the database
