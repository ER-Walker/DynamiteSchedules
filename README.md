# DynamiteSchedules
Web Programming Final for Spring 2026


Basic Project Overview:
- Student input degree and their progress
      ~ selection from list
      ~ only show classes eligible by prereq completions
      ~ build schedule based on mock times
      ~ displays what to anticipate in the future
  - Feature to show tree of prereqs
      ~ indication to finished classes and required classes
  - Login Page
      ~ Save progress over sessions
      ~ admin prielages to override:
            - update prereqs and offered classes
            - if time allows: change individual schedules and overried requirements for students
  - Resources page with professors, the registrar, and tutor resources

  How to SSH
    - Open either a Command Prompt or Terminal from VSC
    - Run the command "ssh provided-user@provided-ip" with your provided information
    - On first attempt, you may need to enter yes to a prompt
    - You will be prompted to enter your password for the user account
    - You should now be in the server. To leave, type exit.
